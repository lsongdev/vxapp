## vxapp project

> wechat mini-program framework

### Features

+ @vxapp/cli 提供创建小程序项目、小程序开发者工具登录、预览、发布
+ @vxapp/wxjs 支持打包构建基于NPM包的小程序代码
+ @vxapp/wxml 提供对微信小程序的 WXML 模版 AST 解析、遍历、转换
+ @vxapp/wxss 提供对微信小程序的 WXSS 样式文件进行依赖分析、转换
